# Sentiment Analysis Project

This project conducts a basic sentiment analysis on text data. It uses NLTK's VADER analyzer if available, with a rule-based fallback system for restricted environments.

## Files
- `sentiment_analysis.py`: Main script to run the analysis.
- `requirements.txt`: List of Python dependencies.

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the script:
   ```bash
   python3 sentiment_analysis.py
   ```
